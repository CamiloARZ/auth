# Instrucciones

## Creando y configurando un nuevo proyecto de Laravel ✒️

**Crera nuevo proyecto**
```
$ composer create-project laravel/laravel auth "5.7.*"
```
**crear la base de datos en Mysql**
```PowerShell
$ mysql -uroot -p
Enter password: 
 
MariaDB [(none)]> CREATE DATABASE auth;
```
